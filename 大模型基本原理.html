云同步设置 · GitHub Gist

📖 怎样获取 Token 和 Gist？
  1. 打开 https://github.com/settings/tokens?type=beta（或 头像 → Settings → Developer settings → Personal access tokens → Fine-grained tokens）
  2. 点 Generate new token
  3. Token name：随便写，比如 llm-map-notes
  4. Expiration：推荐 1 year（到期需重新生成）
  5. Repository access：选 Public Repositories (read-only) 即可（这一项不影响 Gist）
  6. Permissions → Account permissions（不是 Repository permissions）→ 找到 Gists → 改为 Read and write，其他权限全部留 No access
  7. 点 Generate token，复制出现的字符串（形如 github_pat_xxx）。⚠ 离开页面后再也看不到，必须立刻贴进本工具
  8. 把 token 贴到上面 Token 框，Gist ID 留空，点「保存并同步」→ 自动创建一个私有 Gist，右上角圆点变绿即同步成功
  9. 换设备/换浏览器：同样打开本页面 → ⚙ → 贴同一个 token → Gist ID 去 https://gist.github.com 找名为「LLM 知识地图 · 高亮与笔记」的那个，URL 末尾就是 ID → 保存 → 自动拉取之前的数据
